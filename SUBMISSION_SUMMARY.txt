                         HOW TO EVALUATE THE PROJECT

STEP 1: START THE SERVER
  $ npm install
  $ npm run dev
  → Server should start on http://localhost:3000

STEP 2: RUN AUTOMATED TESTS
  $ chmod +x API_TESTS.sh
  $ ./API_TESTS.sh
  → Should show: 19 tests PASSED (100% success rate)

STEP 3: TEST WITH POSTMAN (Optional but recommended)
  - Import: Fastor_CRM_Collection.json
  - Environment: Fastor Local
  - Get a JWT token from Login endpoint
  - Inject token in environment variable: jwt_token
  - Run all 6 endpoints
  → All should return correct status codes (201, 200, 200, 200, 200, 200/409)

STEP 4: VERIFY FUNCTIONALITY
